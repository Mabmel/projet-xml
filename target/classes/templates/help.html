Liste des opérations:

Liste des CV Format XML

  • URL: Liste des cv format xml
  • Méthode attendue: GET
  • Opération: Affiche la liste des CV stockés au format XML.
  • Retour: Flux XML contenant la synthèse des informations pour chaque CV.

Liste des CV Format HTML

  • URL: Liste des CVs
  • Méthode attendue: GET
  • Opération: Affiche la liste des CV stockés au format HTML.
  • Retour: Page HTML présentant les informations de chaque CV.

Détail d’un CV Format XML

  • URL:
  • Méthode attendue: GET
  • Opération: Affiche le contenu complet du CV spécifié par son identifiant au format XML.
  • Retour: Flux XML conforme au schéma xsd cv24, comprenant un seul CV.

Détail d’un CV Format HTML

  • URL:
  • Méthode attendue: GET
  • Opération: Affiche le contenu complet du CV spécifié par son identifiant au format HTML.
  • Retour: Page HTML présentant les informations du CV spécifié.

Ajout d’un CV

  • AJOUTER UN CV
  • Méthode attendue: GET
  • Opération: Ajoute un nouveau CV à la base de données.
  • Retour: Flux XML contenant les informations sur l'opération.

Suppression d’un CV

  • URL:
  • Méthode attendue: GET
  • Opération: Supprime un CV spécifié par son identifiant.
  • Retour: Flux XML contenant les informations sur l'opération.
